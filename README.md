# ADLIYA - O'zbekiston Respublikasi Adliya vazirligi
## Yoshlar Monitoringi Tizimi

### Loyiha haqida
Bu loyiha O'zbekiston Respublikasi Adliya vazirligi uchun yoshlar monitoringi va nazorat tizimi bo'lib, HTML5, CSS3 va Vanilla JavaScript asosida yaratilgan.

### Texnologiyalar
- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- Chart.js (Grafiklar uchun)
- Leaflet.js (Xarita uchun)
- Font Awesome (Ikonlar uchun)

### Sahifalar ro'yxati (16 ta)
1. `login.html` - Asosiy kirish sahifasi
2. `sms-verification.html` - SMS tasdiqlash
3. `password-reset.html` - Parol yangilash
4. `success.html` - Muvaffaqiyatli xabar
5. `dashboard.html` - Asosiy boshqaruv paneli
6. `ideal-mahalla.html` - Ideal mahalla statistikasi
7. `murojaatlar.html` - Murojaatlar bo'limi
8. `ishsizlar.html` - Jami ishsizlar soni
9. `ogir-toifa.html` - O'gir toifadagi yoshlar
10. `migratsiya.html` - Migratsiyadagi yoshlar
11. `grid-view.html` - Grid ko'rinish
12. `profile-main.html` - Profil asosiy
13. `profile-family.html` - Oilaviy ahvoli
14. `profile-relatives.html` - Yaqinlari haqida
15. `murojaatlar-full.html` - To'liq MFY kartlari
16. `murojaatlar-scrolled.html` - Scroll qilingan ko'rinish

### Foydalanish
1. Barcha fayllarni yuklab oling
2. `login.html` faylini brauzerda oching
3. Sahifalar orasida navigatsiya qiling
